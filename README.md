## Programa de envio de archivos mediante ***Taildrop*** de ***Tailscale***

Razones de creacion
---
En lo personal, se me hace un tanto tedioso el enviar archivos desde mi computadora personal linux a algun dispositivo windows o incluso android mediante taliscale.

Pero con este programa solo es necesario ingresar:
  - Ruta del archivo
  - Dispositivo destino

Es un tanto tedioso tambien, pero aqui no necesito aprender los comandos para enviar y aparte me muestra en pantalla:
  - Informacion del dispositivo actual
  - Lista de dispositivos activos
  - Direccion ip de cada uno
  - Sistema operativo de cada uno

Y esto es una ventaja a la hora de digitar los datos de envio

Requerimientos:
---
  - Es necesario tener un tunel VPN de `Tailscale` activo
  - Este programa solo funciona en sistemas operativos `Linux`
  - Se necesita `sudo` para ejecutar el envio de archivos con exito
